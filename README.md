# AcousticCalcs
Set of R functions for common calculations in acoustics and audiometry

```
Installation:
library(devtools)
devtools::install_github("jzeyl/AcousticCalcs", force = TRUE)
```

Functions:
>dBtoPa()

>PatodB()

>transmissionloss()

>phaseshift()
